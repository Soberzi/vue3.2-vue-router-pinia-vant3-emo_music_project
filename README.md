# 先去网易云node.js官网下载好后端包
官网：https://binaryify.github.io/NeteaseCloudMusicApi/#/?id=neteasecloudmusicapi
## 将后端接口开启
node ./app.js

# 安装依赖
npm i

# 将项目跑起来
npm run serve
